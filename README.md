## 🌌 About the Game

Space Cafe Manager is a browser-based idle game built with pure HTML, CSS, and JavaScript — no frameworks, no build tools. You manage a growing space cafe, serve alien customers, research new menu items, hire staff, and prestige your way to galactic fame.

This game was built using **vibe coding** — rapidly prototyping and iterating with AI assistance to bring the idea to life.

---

## ✨ Features

- ☕ **Multiple Workstations** — Espresso, Smoothie, Pastry, Nebula Tea, and Galaxy Cake
- 👽 **Alien Customers** — Ships arrive with orders; serve them before their patience runs out!
- 🤖 **Automation** — Hire a Station Manager and Serving Droid to run the cafe for you
- 🔮 **Prestige System** — Reset for Dark Matter and unlock permanent upgrades
- ⚡ **Space Events** — Random events like Meteor Showers, VIP Customers, and Galactic Rushes
- 🏆 **Achievements** — 9+ milestones to unlock and track
- 🌟 **Animated Starfield** — Canvas-based background with twinkling stars
- 🏠 **Cafe Level** — 10 levels from "Tiny Booth" to "Universal HQ"
- 🎁 **Daily Bonus** — Return each day to claim free Dark Matter
- 💤 **Offline Income** — Earn credits even while the game is closed (up to 8 hours)
- 📱 **Mobile Friendly** — Responsive layout with a bottom navigation bar

---

## 🛠️ Built With

- **HTML5** — Single-file game (index.html)
- **CSS3** — Animations, grid layout, responsive design
- **JavaScript** — Game loop, state management, localStorage save/load
- **Tone.js** — Sound effects
- **Tailwind CSS** — Utility classes for layout
- **Vibe Coding** — AI-assisted rapid development

## 🎮 How to Play

1. **Click a workstation** at the bottom to start making a product
2. **Wait** for it to glow — that means it's ready!
3. **Drag** the glowing workstation onto the matching customer ship to earn credits
4. Use credits to **buy upgrades** from the side panel
5. **Prestige** when you have enough to earn Dark Matter for permanent boosts

---

## 🚀 Running Locally

No installation needed. Just open `index.html` in any modern browser:

```
double-click index.html
```

Or serve it with a local server:

```bash
python -m http.server 8080
# then open http://localhost:8080
```
